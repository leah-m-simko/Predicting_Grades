# Predicting_Grades
Analysis of a dataset containing information from two secondary schools in Portugal about various segments of a student’s life including: social factors, gender, and academic information. The project sets out to answer the following questions:
1.	What personal factors are most influential to a student’s success in secondary school? 
2.	Can we accurately predict a student’s success in a course based on their personal information?
3.	Is there a correlation between alcohol consumption and a student’s success in school?
